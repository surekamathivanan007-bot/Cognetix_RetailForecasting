# Retail Sales Forecasting

Project 7 - Advanced Stage
Internship: Cognetix Technology

## About
Analyzed historical retail sales data and forecasted future sales for next 12 months using ARIMA time series forecasting model.

## Dataset
Download from Kaggle: https://www.kaggle.com/datasets/manjeetsingh/retaildataset
Place all CSV files in the project folder before running the notebook.

## Libraries used
- pandas
- numpy
- matplotlib
- statsmodels

## How to run
pip install pandas numpy matplotlib statsmodels jupyter
jupyter notebook analysis.ipynb
